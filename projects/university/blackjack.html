Blackjack in Python

An implementation of Blackjack in Python 3.

Description

For the very first programming assignment, we were required to write a Python program that allows a player to play a game of Blackjack against the computer. The program allowed a player to play as many games of Blackjack as they wish.

We were given a helper module that included the cards and a playing deck. Additionally, it also included functions to check whether the deck is empty, to reset the deck and to deal one card. We were also given a specific set of rules that our assignments would abide by.

With that, we set off to recreate Blackjack in Python.

Implementation

As I had some prior experience with Python, I went a little overboard with my own assignment. The assignment specified a number of functions we had to implement, such as display_hand, get_hand_total and get_hit_choice. As this was the very first programming assignment, nothing complicated was used. I used lists, dictionaries, loops, subscripting, string concatenation, booleans and more to implement the required functions. After implementing the required functions, it was fairly simple to begin implementing the rules of Blackjack. At the end of the game, the user is prompted whether they would like to play again. If they don't statistics are shown that includes the number of games the user won or lost.
