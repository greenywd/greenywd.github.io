This Website

A portfolio of sorts.

Why?

Long story short, I needed a place to showcase my own projects that I've worked on in my own time. I figured a basic website would work, as it's nothing too complex to create and I already had some experience from University in Web Development.

Technologies

As you can probably tell, this is a fairly basic website. No databases, no crazy UI/UX, Bootstrap is the only framework (aside from it's JavaScript dependencies) used. Not only is it easy to use but I learnt more than the basics in the Web Development class at Uni, so why not put those skills to good use? As I only need static content served, I had no reason to not choose GitHub Pages for hosting. It's free, and with a check of a box, a GitHub repository is being used to serve a website.

Links

As with many of my projects, this website is fully open source and available at GitHub. Not that it matters as you can inspect the source from your browser, but it's the thought that counts, right?
